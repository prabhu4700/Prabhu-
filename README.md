

This project is a simple Python program that explains the concept of reciprocity using basic programming. Reciprocity means mutual help or give-and-take between people. In this program, users help each other, and the system tracks how much help is given and received.

The program uses a class-based approach to represent users. Each user can help another user, and the program updates their help records. At the end, it shows the total help given and received by each user.

This project is useful for beginners who want to learn Python concepts like classes, objects, functions, and basic logic in a simple and meaningful way.
